# Grup Gerak Khas — 3D Tactical Tour Website

A premium, interactive 3D tactical tour website for **Grup Gerak Khas** (Malaysian Army Special Forces). 

Built as a pure, zero-build static site using **HTML5**, **CSS3**, **Vanilla JavaScript**, and **Three.js** to guarantee absolute compatibility when opened directly as a file (`file:///`) or hosted on static pages like **GitHub Pages**.

## ⚔️ Key Features
- **Interactive 3D Scene**: Custom wireframe tactical globe, sweeping radar scanner, pulsing communication beacon, and particles.
- **Scroll-Driven Camera**: Smoothly animates and moves the Three.js viewport between sections as you scroll.
- **Dual Language**: Seamless instant toggle between English and Bahasa Melayu (fully client-side).
- **Responsive Layout**: Designed mobile-first, supporting all viewports with a glassmorphism theme.
- **Zero-Build Architecture**: Simply double-click `index.html` to run locally! No compilation steps or local dev servers required.

## 🚀 Running Locally
You don't need `npm` or any server tools.
1. Open the folder.
2. Double click **`index.html`** in your browser.

## 🌐 Deploying to GitHub Pages
To host on `https://ggk2020.github.io/website/`:
1. Push this entire repository root to your `main` branch.
2. Go to your repository settings: **Settings → Pages**.
3. Under **Build and deployment → Source**, select **Deploy from a branch**.
4. Set the branch to **`main`** and the directory to **`/ (root)`**, then click **Save**.
5. The site will deploy instantly.
