# Grup Gerak Khas Website

Welcome to the **Grup Gerak Khas** website project! This project aims to create a modern and stylish website for the organization, featuring a dual language option (Bahasa Melayu and English) and incorporating fancy emojis for a vibrant user experience.

## Project Structure

The project is organized as follows:

```
grup-gerak-khas-website
├── src
│   ├── index.html          # Main HTML document
│   ├── styles              # CSS styles
│   │   └── main.css        # Main stylesheet
│   ├── scripts             # JavaScript files
│   │   └── main.js         # Main script for interactivity
│   ├── locales             # Language translations
│   │   ├── ms.json         # Bahasa Melayu translations
│   │   └── en.json         # English translations
│   ├── components          # Reusable HTML components
│   │   ├── header.html     # Header component
│   │   ├── footer.html     # Footer component
│   │   └── hero.html       # Hero section component
│   └── data                # Structured data
│       └── info.json       # Organization information
├── package.json            # NPM configuration
├── .gitignore              # Git ignore file
└── README.md               # Project documentation
```

## Features

- **Dual Language Support**: Users can switch between Bahasa Melayu and English seamlessly.
- **Modern Design**: The website is styled with a contemporary aesthetic to enhance user engagement.
- **Dynamic Content**: Information about the organization is dynamically loaded from structured data.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd grup-gerak-khas-website
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Project**:
   You can use a local server to view the website. For example, you can use `live-server` or any other static server of your choice.

4. **Open in Browser**:
   Navigate to `http://localhost:PORT` (replace PORT with the appropriate port number) to view the website.
